# Project Organizer Application

## To setup database:
  + You need to have local PostgreSQL database
  + Inserts database credentials into /backend/.env in lines 1-5.
  + Run commands:
    - cd backend/
    - npm run migrations:up
  + Done!

## To start backend:
  + Run commands:
    - cd backend/
    - npm start
  + Done!

## To start backend:
  + Run commands:
    - cd frontend/
    - npm run dev
  + Go to http://localhost:3000/home
  + Done!
